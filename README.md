# Portfolio Site

This is a minimal static portfolio site you can edit quickly.

- Edit the HTML: `index.html` — change your name and tagline.
- Edit styles: `styles.css` — tweak variables at the top (colors, radius).
- Edit JS: `script.js` — small helpers (theme toggle, year) and project rendering.
- Edit projects: `data/projects.json` — add or update projects; the page renders this file into the Projects section.

Preview locally by opening `index.html` in a browser or using a simple static server:

```powershell
# from this folder
python -m http.server 8000
# then open http://localhost:8000
```
